1. Написать хранимую процедуру, выводящую на экран список студентов, до сих пор не сдавших книги. 

2. Написать хранимую процедуру, подсчитывающую факториал переданного числа. (5! = 1*2*3*4*5 = 120) (0! = 1) (факториала отрицательного числа не существует).

3. Написать хранимую процедуру, возвращающую количество студентов, которые никогда не брали книги в библиотеке. 

4. Написать хранимую процедуру, возвращающую количество взятых книг по каждой из групп и по каждой из кафедр (departments). 

5. Написать хранимую процедуру, возвращающую список книг, отвечающих набору критериев. Критерии: имя автора, фамилия автора, тематика, категория. Кроме того, список должен быть отсортирован по номеру поля, указанному в 5-м параметре, в направлении, указанном в 6-м параметре.

6. Написать хранимую процедуру, которая возвращает список библиотекарей и количество выданных каждым из них книг.

7. Создать ХП, которая вернёт имя и фамилию студента, набравшего наибольшее количество книг.

8. Создание хранимой процедуры, которая вернёт количество взятых из библиотеки книг и преподавателями, и студентами.
